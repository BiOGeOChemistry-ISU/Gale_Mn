# Gale_Mn
codes for plotting and data analysis related to Gale crater (Mars) Mn enrichments and Mn enrichments in terrestrial lacustrine sediments

Data source is https://zenodo.org/records/15425479

Start in Gale_Mn with Preparing_data.R to convert the data downloaded to a useable format, and merge with the categorical labels provided in this folder.

Then use the produced file in the subfolders to generate analyses and plots described in the manuscript by Ledesma et al.
